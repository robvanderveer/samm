This folder contains the template files for the SAMM v2 document model data.

Please do not modify these template files, but copy the contents into a new file and adapt them there.

Most content should be self-explanatory.
Identifiers are based on GUIDs. This ensures that the references are not too sensitive to different change scenarios.
You can use the following site to generate uniaue GUIDs: https://www.guidgenerator.com/online-guid-generator.aspx
Uncheck the "hyphens" default setting.

Kindly provide any feedback to bart.dewin@owasp.org (or create a Github issue on the specific template file(s)).
